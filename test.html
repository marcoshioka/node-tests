API de estados e cidades do Brasil

Esta é uma API que permite buscar pelos estados e cidades do Brasil.

Rotas:

/estados: retorna a listagem com todos os estados do Brasil

/estados/{siglaEstado}: retorna o estado pesquisado e suas informações

/estados/{siglaEstado}/cidades: retorna a listagem das cidades do estado pesquisado

/estados/{siglaEstado}/cidades/{nomeCidade}: retorna a cidade do estado pesquisado e suas informações
